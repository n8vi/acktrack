The ACKTRACK library

This is the home of the ACKTRACK library, useful for watching acknowledgements to TCP sessions your software is engaged in. You can initialize an ACK tracker by passing a socket handle, or by passing strings describing each end of the connection. The library is cross-platform, and is tested on Linux and Windows using gitlab CI. Documentation is not complete yet, which is to say that for the most part, the documentation is the fairly messy source code. That will be cleaned up. However ...

The funding for this project is temporarily suspended due to COVID-19. While I intend to continue work on the library, I can make no guarantees of how quickly updates will occur.
